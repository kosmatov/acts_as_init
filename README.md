# ActsAsInit

Gem call method 'init' on new AR model

## Installation

Add this line to your application's Gemfile:

    gem 'acts_as_init', :git => 'git://github.com/kosmatov/acts_as_init.git'

And then execute:

    $ bundle

## Usage

Define method 'init' in your model.
